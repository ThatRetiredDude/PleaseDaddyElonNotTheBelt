# Tested version matrix (informal)

This project does **not** use a full lock file; [requirements.txt](requirements.txt) and CI on Python **3.10, 3.12, 3.13** are the main guardrails. When upgrading **tweepy**, re-run the full “Fetch Newer/Older / Test User Auth / delete queue smoke” on your own account before you rely on it in production.

See also [ARCHITECTURE.md](ARCHITECTURE.md).
